CSS Colors

  • Can be used to decorate the pages by using colors

1. CSS Background Color:

  • Used to style background of content.

2. CSS Text Color

  • It is used to define color to the text.

3. CSS border Color

  • It is used to define border.

4. CSS RGB Colors:

  • Each parameter (red, green, blue) defines the intensity of the color between 0 to 255.

4. CSS RGBA Colors:

  • RGBA color values are an extension of RGB color values with an alpha channel - which specifies the opacity for a color.
  • The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all)

5. HEX Values:

  • HEX color values are six-digit color values, which are typically prefixed with "#"

6. 3-Digit Hex Values:

  • 3-digit hex code is a shorthand for some 6-digit code.
  • 3-digit code are in '#rgb' form

HSL and HSLA Colors:

  • HSL is a combination of hue, saturation and lightness (HSL).
  • Hue is a degree of color wheel from 0 to 360. 0 is red, 120 is green, 240 is blue.
  • saturation is a percentage value. 0% means a shade of gray, and 100% is the full color.
  • Lightness is also a percentage. 0% is black, 50% is neither light or dark, 100% is white
  • Example of hsl: hsl(0, 100%, 50%)
  • Example of hsla: hsla(9, 100%, 64%, 0.8)
