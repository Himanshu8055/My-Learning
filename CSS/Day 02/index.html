How To Add CSS

Three Ways to Insert CSS :

  1. External
  2. Internal
  3. Inline

1. External CSS

  • Can change entire design by changing single file of CSS
  • Easy to understand if other part is seperated like css and javasc.

2. Internal CSS

  • In this, style is used in single page
  • It is written in <style> tag.

3. Inline CSS

  • Style is used inside the element
  • It is used by writing style attribute of the opening tag.
