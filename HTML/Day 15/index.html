    • HTML Encoding (Character Sets)

    HTML encoding is used to display special characters in HTML documents.

    The ASCII Character Set

    The ASCII character set is a character encoding standard for electronic communication.

    1. Special characters from 128 to 159
    2. Identical to UTF-8 from 160 to 255
    3. Identical to ASCII for the first 127 characters
